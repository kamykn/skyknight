# Install
## Plug
Installation with Plug

```
Plug 'kmszk/skyknight'
```

## NeoBundle
Installation with NeoBundle

```
NeoBundle 'kmszk/skyknight'
```

# Setting
Add to your .vimrcbe:

```
colorscheme skyknight
```
